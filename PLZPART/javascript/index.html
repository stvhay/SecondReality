Second Reality PLZ plasma JavaScript port

Live demo Source and notes Repository

This recreates the plasma part listed in SCRIPT immediately before Plasmacube. It keeps the original lookup-table formula, phase speeds, palette fades, 70 Hz stepping, VGA planar checkerboard interleave, and the 320x400 line-compare presentation. The default view sends that signal through a WebGL2 shader modeled on an NEC MultiSync-class 1992 shadow-mask monitor: CRT-gamma decode, vertical scan beam, staggered shadow-mask triads, phosphor spread, and separable halation bloom. Geometry is left flat — no curvature, no warp.
